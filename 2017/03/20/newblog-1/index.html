实验一 在Github上用Markdown创建技术博客

一、实验目的

1.熟悉github网站

2.掌握Markdown语法

二、实验任务

1.注册Github账号

2.创建Gihub Pages repository

3.发布第一篇博客

4.学习Markdown

5.利用Hexo生成博客并部署到Github

三、实验步骤与结果

1.注册Github账号

登入http://github.com, 注册github账号

我的github账号用户名为jakieenchan

2.创建Gihub Pages repository

3.发布第一篇博客

⑴本地配置hexo

1.安装Git

2.安装Node

3.设置npm镜像为淘宝

npm config set registry https:// registry.npm.taobao.org

4.安装hexo

npm install –g hexo-cli

5.初始化站点

hexo init myhexo

cd myhexo

npm install

6.浏览本地站点

hexo server

⑵在github上部署hexo

1.在Github上新建repository，名称是用户名.github.io(即jakieenchan.github.io)

2.修改配置文件_config.yml的deploy部分，注意空格

type: git

repository: https://github.com/jakieenchan/jakieenchan.github.io.git branch: master

修改配置文_config.yml的site部分

Site

title: jakieenchan

subtitle:

description:

author: jakieenchan

language: zh-CN

timezone: Asia/Shanghai

3.安装 hexo-deployer-git

npm install hexo-deployer-git –save

4.配置git全局邮箱与名字

git config –global user.email 1054636188@qq.com

git config –global user.name “jakieenchan”

5.生成博客

hexo generate

6.部署博客

hexo deploy

⑶添加新文章

1.生成新文章

hexo new newblog

2.用编辑器修改文章：myhexo\source_posts\newblog.md

3.生成并发布到Github

hexo d -g

四、实验小结

通过本次实验充分理解了github，了解了Markdown文件，更深一步的掌握了搭建博客平台的技术。
